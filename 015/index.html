Hello World!

Lab Topics:
File structure, code editor, ftp, vpn, basic html.

In-Class Activity:
Create a basic one page html document, saved as hello-world.html, that explores the use of basic HTML tags. Upload the file to your creative server.

Notes from Class:
Connecting to the CU Creative Server from off-campus requires two steps:
1) Connect to CU's VPN
2) Connect via FTP (using either Fetch or Cyberduck)

The FIRST time you connect to the CU VPN (using Cisco AnyConnect Secure Mobility Client), you should enter vpn.colorado.edu into the address bar. It will then prompt you to enter your identikey credentials. Every time after that you will connect to UCB VPN.

Check these step-by-step instructions for uploading files to the Creative Server.
